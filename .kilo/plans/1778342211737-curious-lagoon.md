# Portfolio Section Restructuring - Corporate Projects Visibility

## Problem
Corporate projects (white label work) are hidden behind a tab that users might not notice, potentially causing them to miss viewing 4 additional high-quality projects.

## Current State
- Portfolio has two sections: Retail (4 projects) and Corporate (4 locked/NDA projects)
- Section switcher is in the top-right with two tabs: "Retail" and "Corporate"
- Corporate tab has a small lock icon but may not be prominent enough
- Once switched, projects require "hold to decrypt" interaction

## Proposed Solutions

### Option 1: Enhanced Visual Prominence (Recommended)
Make the corporate button impossible to miss through multiple visual cues:
- Add animated glow/pulse effect to corporate tab
- Add project count badges: "Retail (4)" and "Corporate (4)"
- Increase lock icon size and add pulsing animation
- Add hover tooltip: "View 4 white-label projects"
- Change tab styling to be more distinct (e.g., darker/different color for corporate)

**File changes:**
- `components/Portfolio.tsx:461-491` - Section tabs area

### Option 2: Teaser Banner
Add a small banner/chip below retail projects that teases corporate work:
- Position: Below the last retail project or above footer
- Text: "🔒 View 4 additional corporate projects (white-label work)"
- Clicking scrolls to top and switches to corporate tab
- Can be dismissible after first interaction

**File changes:**
- `components/Portfolio.tsx:494-774` - Add banner in main showcase area

### Option 3: Inline Preview
Show a blurred preview of 1-2 corporate projects at the bottom of retail section:
- Display 1-2 locked corporate project cards in a separate row
- Labeled "Corporate Projects - Click to Unlock"
- Clicking switches to full corporate view
- Creates curiosity through partial reveal

**File changes:**
- `components/Portfolio.tsx:494-774` - Add preview cards in main layout

### Option 4: Stats Counter with Call-to-Action
Add a stats panel near the header showing total project breakdown:
- "8 Total Projects: 4 Retail | 4 Corporate (NDA)"
- Make the corporate portion clickable
- Add subtle animation to draw eye

**File changes:**
- `components/Portfolio.tsx:440-492` - Header section

### Option 5: Auto-rotate Hint
After viewing all retail projects (auto-advance completes full cycle):
- Show a modal/tooltip suggesting: "You've seen all retail projects. View 4 corporate projects?"
- With "Show Corporate" and "Dismiss" buttons
- Only show once per session

**File changes:**
- `components/Portfolio.tsx:330-341` - Auto-advance logic
- Add modal/tooltip component

### Option 6: Split Toggle Button
Replace the subtle tabs with a larger, more prominent toggle switch:
- Larger buttons with icons and descriptions
- "Retail Projects (4)" vs "Corporate Vault (4)"
- iOS-style toggle with sliding animation
- More prominent positioning

**File changes:**
- `components/Portfolio.tsx:460-492` - Redesign section switcher

## Selected Approach

**Option 1: Enhanced Visual Prominence**

User selected enhanced tabs with visual cues to make corporate section discoverable while maintaining the current UX structure.

## Implementation Plan

### 1. Add New Keyframe Animation
**Location:** `components/Portfolio.tsx:375-382` (style block)

Add CSS animation for corporate tab pulse effect:
```tsx
@keyframes corporatePulse { 
  0%, 100% { 
    box-shadow: 0 0 0 0 rgba(242,92,67,0.4);
    transform: scale(1);
  }
  50% { 
    box-shadow: 0 0 12px rgba(242,92,67,0.3);
    transform: scale(1.02);
  }
}
```

### 2. Enhance Section Tab Buttons
**Location:** `components/Portfolio.tsx:469-490` (tab mapping section)

Changes to implement:

1. **Add project count badges** to both tabs:
   - Display format: "Retail (4)" and "Corporate (4)"
   - Makes it clear there are more projects to see

2. **Increase lock icon size**:
   - Change from 10x10 to 12x12
   - Add subtle glow filter when retail section active
   - Current: `<svg width="10" height="10">`
   - New: `<svg width="12" height="12" style={{ filter: section === 'retail' ? 'drop-shadow(0 0 4px rgba(242,92,67,0.6))' : 'none' }}`

3. **Add pulsing animation** to corporate button when retail is active:
   - Only animate when user is viewing retail section
   - Stops animating once they've visited corporate
   - Add to button style: `animation: section === 'retail' ? 'corporatePulse 2.5s ease-in-out infinite' : 'none'`

4. **Add hover tooltip** using title attribute:
   - "View 4 white-label projects" for corporate tab
   - Simple, non-intrusive way to explain what's there

5. **Enhance visual distinction**:
   - Slightly brighter glow on corporate tab background when inactive
   - Make the difference between retail/corporate more pronounced

### 3. Update Tab Rendering Logic

Replace the current tab mapping code with enhanced version:

```tsx
{[
  { key: "retail", label: "Retail", count: RETAIL.length, tooltip: "Client projects" },
  { key: "corporate", label: "Corporate", count: CORPORATE.length, tooltip: "White-label & NDA projects" },
].map(t => (
  <button 
    key={t.key} 
    onClick={() => switchSection(t.key)} 
    title={t.tooltip}
    style={{
      padding: "0 24px", height: 44,
      fontFamily: "var(--font-mono)", fontSize: 10, letterSpacing: "0.2em",
      textTransform: "uppercase", border: "none", background: "transparent",
      cursor: "pointer",
      color: section === t.key ? (t.key === "corporate" ? "#F25C43" : "#fff") : "rgba(255,255,255,0.35)",
      display: "flex", alignItems: "center", gap: 8,
      position: "relative", zIndex: 2, whiteSpace: "nowrap",
      transition: "color 0.3s, transform 0.2s",
      // Add pulse animation to corporate tab when retail is active
      animation: (t.key === "corporate" && section === "retail") 
        ? "corporatePulse 2.5s ease-in-out infinite" 
        : "none",
    }}
  >
    {t.key === "corporate" && (
      <svg 
        width="12" 
        height="12" 
        viewBox="0 0 24 24" 
        fill="none" 
        stroke="currentColor" 
        strokeWidth="2"
        style={{
          filter: section === 'retail' 
            ? 'drop-shadow(0 0 4px rgba(242,92,67,0.6))' 
            : 'none',
          transition: 'filter 0.3s',
        }}
      >
        {revealed ? (
          <><rect x="3" y="11" width="18" height="11" rx="1"/><path d="M7 11V7a5 5 0 0110 0"/></>
        ) : (
          <><rect x="3" y="11" width="18" height="11" rx="1"/><path d="M7 11V7a5 5 0 0110 0v4"/></>
        )}
      </svg>
    )}
    {t.label} ({t.count})
  </button>
))}
```
